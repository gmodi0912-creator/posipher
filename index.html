Challenge & Insight

Reso Health sought to bridge the gap between complex health diagnostics and hardware-driven nutrition delivery. The core challenge lay in creating an intuitive mobile gateway that translated personalized assessment rules into real-world machine dispensing via NFC pairing.

  • Personalization Complexity: Mapping lifestyle assessment data into precision supplement plans via a custom rules-engine.
  • Secure IoT Pairing: Managing time-bound machine sessions tied to specific orders using iOS/Android NFC protocols.
  • Trust-First UX: Maintaining healthcare-grade clarity and accessible design to reduce user anxiety during hardware interaction.

Engineering the Solution

NFC Session Binding

Robust hardware-handshake that initiates a 60-second secure dispense window, preventing unauthorized machine access.

Real-Time IoT Logic

Live dispense tracking system that monitors machine state (Started → Dispensing → Completed) with fail-safe error handling.
